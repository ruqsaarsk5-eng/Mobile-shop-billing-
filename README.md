📱 Mobile Shop Billing System (Dart CLI)
A command-line Mobile Shop Billing System built using Dart. This program allows users to select products, enter quantity, and generate a bill.
🚀 Features
Multiple product selection
Loop-based menu system
Quantity input
Add multiple items
Final billing system

🛠️ Tech Used
Dart (CLI)

▶️ How to Run
Open terminal inside project folder
Run:

dart run

Enter:
Customer name
Product choice
Quantity
Repeat until finished

📂 Project Structure
bin/mobile_shop.dart → Main logic
pubspec.yaml → Configuration

🎯 Purpose
This project is made for learning:
Loops (while / do-while)
Switch case
Lists
User input handling
Billing logic
